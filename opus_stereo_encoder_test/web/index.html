Opus Stereo RTP Stream Test

This page controls the C++ RTP streamer. Click the button to start transmitting Opus over RTP and play back a WAV representation of the same PCM16 source.

Idle

How it works

  1. Send a POST /start request to the local server.
  2. The server encodes stereo PCM16 into Opus frames and pushes them over RTP.
  3. The browser downloads a WAV rendering of the same source for direct playback.
